The general structure of this anatomy:

Propositional Logic.

Def. The fundamental concept is a proposition: a variable that can take on the values true, or false. PL is responsible for deciding the truth value of a proposition, and thus deals with the whole thing, under the assumption that it is true or false.

Def. A proposition which always true is a tautology, a proposition which is always false is a contradiction. We can build new propositions, out of old propositions, using logical operators, the simplest of which are unary, and consume one proposition, to return another.

Unary & Binary Operators:

The binary operator for implication, is tricky: the reason that false can imply true is based on the latin phrase, "ex falso quod libet", which means from false assumptions, anything follows, and the mathematical benefit is that this definition, unlocks proof by contradiction, as: P ⟹ Q↔¬Q ⟹ ¬P\\P \implies Q \leftrightarrow \neg Q \implies \neg PP⟹Q↔¬Q⟹¬P.

Theorem: Let p,qp,qp,q be propositions. Then: (p ⟹ q)⟺((¬q)⟹(¬p))\\ (p \implies q) \Longleftrightarrow ((\neg q) \Longrightarrow (\neg p))(p⟹q)⟺((¬q)⟹(¬p))

Remark 1\text{Remark}\;1Remark1: With these operators, we agree on the decreasing binding strength of the operators in the following sequence: ¬,∧,∨, ⟹ ,⟺\neg, \land, \lor, \implies, \Longleftrightarrow¬,∧,∨,⟹,⟺.

Remark 2\text{Remark}\; 2Remark2: All higher order operators, of the form σ(p1,...,Pn)\sigma(p_1,...,P_n)σ(p1​,...,Pn​), can be constructed from the nand operator, denoted: (⊼\barwedge⊼).

Predicate Logic.

Def. The fundamental concept, a predicate, informally, is a proposition valued function of some variable(s). For instance P(x)P(x)P(x) is true or false depending on x.

Def. A predicate of two variables is a relation.

P(x)P(x)P(x) is a proposition for choices of xxx, and similarly Q(x,y)Q(x, y)Q(x,y) is such a proposition with a truth value dependent on xxx, and yyy. Predicate logic doesn't examine how predicates are composed, as that requires language for establishing the rules for combining xxx and yyy into a predicate. We also do not specify which set they come from, as we're building up to define the set as a concept, and thus can't make use of it prior.

We can still build predicates from given ones, i.e Q(x,y,z):⟺P(x)∧R(y,z)Q(x,y,z): \Longleftrightarrow P(x) \land R(y,z)Q(x,y,z):⟺P(x)∧R(y,z), and we can construct new propositions from predicates using quantifiers.

Quantifiers:

Def. Universal Quantifier (∀\forall∀):

Def. Let P(x)P(x)P(x) be a predicate. Then ∀x:P(x)\forall x : P(x)∀x:P(x), and this is defined to be true if P(x)P(x)P(x) is true independent of xxx.

Def. Existential Quantifier: (∃\exists∃):

Let P(x)P(x)P(x) be a predicate. Then ∃x:P(x):⟺¬(∀x:¬P(x)) \exists x : P(x) :\Longleftrightarrow \neg (\forall x : \neg P(x))∃x:P(x):⟺¬(∀x:¬P(x))

Corollary: ∀x:P(x)⟺¬(∃x:P(x))\forall x : P(x) \Longleftrightarrow \neg (\exists x : P(x))∀x:P(x)⟺¬(∃x:P(x)).

If, for every xxx, it is true that xxx has some property, then it must not be true that there exists an xxx which doesn't.

Remark 1\text{Remark}\;1Remark1: Quantification allows for predicates of multiple variables: Given some predicate P(x,y)P(x,y)P(x,y), then for fixed values of y,P(x,y)y, P(x,y)y,P(x,y) is a predicate of one variable, and we define Q(y):⟺∀x:P(x,y)Q(y) : \Longleftrightarrow \forall x : P(x, y)Q(y):⟺∀x:P(x,y). Now we can define ∃y:∀x:P(x,y):⟺∃y:Q(y)\\ \exists y: \forall x : P(x, y) :\Longleftrightarrow \exists y : Q(y)∃y:∀x:P(x,y):⟺∃y:Q(y)

Here, yyy is a bound variable, and xxx is free.

Remark 2\text{Remark}\;2Remark2: Order matters. ∀x:∃y:P(x,y)\forall x : \exists y: P(x,y)∀x:∃y:P(x,y) is generically different from ∃y:∀x:P(x,y)\exists y : \forall x : P(x,y)∃y:∀x:P(x,y).

Consider the proposition expressing the existence of additive inverses in R\mathbb{R}R:

∀x:∃y:x+y=0\forall x : \exists y : x + y = 0∀x:∃y:x+y=0.

This states that for each xxx, there exists a yyy, such that x+y=0x+y=0x+y=0, or −x-x−x. If we swap the quantifiers, i.e ∃y:∀x:x+y=0\exists y: \forall x : x + y = 0∃y:∀x:x+y=0, what we're saying that for every xxx, there is a yyy such that x+y=0x+y=0x+y=0, which is actually false, as if x+y=0,(x+1)+y≠0x + y = 0, (x+1) + y \neq 0x+y=0,(x+1)+y=0, meaning one yyy can't work for even two xxx's, let alone every one.

Axiomatic Systems & Theory of Proofs.

Def. An axiomatic system is a finite sequence of propositions a1,a2,...,ana_1, a_2,..., a_na1​,a2​,...,an​, called axioms.

Def. A proof of a proposition in an axiomatic system a1,a2,...,ana_1, a_2,..., a_na1​,a2​,...,an​, is a finite sequence of propositions q1,q2,...,qnq_1,q_2,...,q_nq1​,q2​,...,qn​, such that for any 1≤j≤M1 \leq j \leq M1≤j≤M, either:

If proposition P\mathcal{P}P is provable within the system, a1,…,ana_1, \ldots, a_na1​,…,an​, we say a1,…,an⊢Pa_1, \ldots, a_n \vdash \mathcal{P}a1​,…,an​⊢P, or this sequence proves P\mathcal{P}P.

Consistency of axiomatic systems depends on there being propositions that weren't provable in the system, due to the issue of ex falso quodlibet, which allows us to prove anything, given the right false starting point, and the fact that things aren't provable relies on there being no contradictory axioms.
