The ∈\in∈ relation: we do not define this symbol, or a set, but rather the associated axioms, telling how they should be used, and then definition is done in terms of these.

There are four axioms regarding existence and the empty set, four regarding constructing new sets, and the axiom of foundation.

The ∈\in∈ relation allows us to define:

x∉y: ⟺ ¬(x∈y)x \notin y :\iff \neg (x \in y)x∈/y:⟺¬(x∈y).

x⊆y: ⟺ ∀a:(a∈x ⟹ a∈y)x \subseteq y :\iff \forall a : ( a \in x \implies a \in y)x⊆y:⟺∀a:(a∈x⟹a∈y).

x=y: ⟺ (x⊆y)∧(y⊆x)x = y :\iff (x \subseteq y) \land (y \subseteq x)x=y:⟺(x⊆y)∧(y⊆x).

x⊂y: ⟺ (x⊆y)∧(̸x=y)x \subset y :\iff (x \subseteq y) \land \not(x = y)x⊂y:⟺(x⊆y)∧(x=y).

Russel's Paradox

Suppose a set UUU, with the property ∀x:(x∉x ⟺ x∈u)\forall x: (x \notin x \iff x \in u)∀x:(x∈/x⟺x∈u).

UUU contains all sets which do not contain themselves, and we want to know if UUU is a set as well. We then consider U∈UU \in UU∈U, which is a proposition, if UUU is a set.

If U∈UU \in UU∈U is true, then ¬(U∈U)\neg (U \in U)¬(U∈U) is true as well, meaning that UUU is not an element of itself, leading us to the contradiction u∈u ⟹ ¬(u∈u)u \in u \implies \neg(u \in u)u∈u⟹¬(u∈u), which indiciates U∈UU \in UU∈U is not a proposition.

However if U∉UU \notin UU∈/U, then we have a contradiction in the other direction, which is u∉u ⟹ ¬(u∉u)u \notin u \implies \neg(u \notin u)u∈/u⟹¬(u∈/u).

This means U∈UU \in UU∈U is not a proposition, as it is neither true, nor false, implying that in fact, UUU is not a set.
